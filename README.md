- React

- TailwindCSS
- React Router

## Features
- Interactive UI
- Realistic mock data
- Recreation of mutliple UI features

## Future Development

## Getting Started
To run this project locally:
- Clone this repo
- Run `npm install` in root directory
- Run `npm run dev` in root directory <br>
View Deployed version here: <br>
